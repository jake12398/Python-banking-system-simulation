# Python-banking-system-simulation
Python-based banking simulation with multi-user accounts, transactions, standing orders, and balance history visualisation using Matplotlib.

## Features
- Multi-user login system with PIN authentication  
- Deposit and withdrawal functionality  
- Transfers between users  
- Transaction history tracking  
- Standing orders (recurring payments)  
- Compound interest applied over time  
- Balance history visualisation using Matplotlib  

## Tech Stack
- Python  
- JSON (data storage)  
- Matplotlib (data visualisation)
